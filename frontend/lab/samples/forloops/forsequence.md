# 🔢 Sequences with `for`

⭐ Write 5 `for` loops that generate the following sequences:

- `0`, `1`, `2`
- `1`, `2`, `3`
- `1`, `4`, `7`
- `8`, `5`, `2`
- `-1`, `-4`, `-7`

### [>] Hint

You only need to find the correct `START`, `STOP` and `STEP` values in:

```evy
for x := range START STOP STEP
  print x
end
```
