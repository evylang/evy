# 🐛 Make a Worm

1. **Run** this program.
2. **Create** some cool output.
3. **Change** this program to draw a circle with
   - ⭐ your favorite **color**
   - ⭐ a slightly different **size**
   - ⭐ a different **outline color**.
     Use the [`stroke`](/docs/builtins.html#stroke) command.
