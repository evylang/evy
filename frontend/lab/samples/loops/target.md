# 🎯 Target board

## ⭐ Draw a target board

Draw a target board with concentric circles.
![Target board](img/target.svg)

### [>] Hint

```evy
//...
while r > ❓
    circle r
    r = ❓ - ❓
end
```

---

## ⭐ Animation

Animate the circles with a `sleep` for 0.2 seconds.

### [>] Hint

```evy
//...
while r > ❓
    // ...
    sleep ❓
end
```

---

## ⭐ Tweak

Increase the starting radius to cover the whole canvas.
